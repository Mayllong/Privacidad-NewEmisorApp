Política de Privacidad para NewEmisorApp

ACTUALIZADO: 28 DE ENERO, 2026

En NewEmisorApp, nos comprometemos a proteger tu privacidad. Esta Política de Privacidad explica qué información recopilamos, cómo la usamos y las opciones que tienes con respecto a tu información cuando usas nuestra aplicación móvil ("la App").

Al usar NewEmisorApp, aceptas las prácticas descritas en esta política.

1. Información que Recopilamos

Para que la aplicación funcione correctamente y para mejorar tu experiencia, recopilamos ciertos tipos de información.

a) Información Recopilada Automáticamente:

  • Datos de Publicidad: Para poder ofrecer la aplicación de forma gratuita, mostramos anuncios a través de Google AdMob. AdMob recopila información para mostrar anuncios relevantes para ti. Esto puede incluir:
    • Identificador de Publicidad de tu dispositivo.
    • Dirección IP.
    • Información no personal sobre tu dispositivo (como el modelo, el sistema operativo y el idioma).
    • Datos sobre tu interacción con los anuncios.
  • Datos de Uso y Diagnóstico (Logs): Cuando usas la App o si ocurre un error, recopilamos automáticamente datos técnicos. No usamos esta información para identificarte personalmente, sino para solucionar problemas.

b) Información de Terceros (Contenido Web):

La App utiliza un componente WebView para mostrar contenido directamente de nuestras páginas web. NewEmisorApp no controla el seguimiento que estas páginas web puedan realizar.

2. Cómo Usamos tu Información

  • Operar la App: Para mostrarte el contenido y los canales de video.
  • Mostrar Anuncios: La monetización nos permite mantener la aplicación gratuita.
  • Mejorar la App: Análisis de errores para mayor estabilidad.
  • Facilitar Actualizaciones: Verificación y descarga de nuevas versiones.
  • Descargar Otras Aplicaciones: Gestión de descargas en la sección "Cargar Apps".

3. Actualizaciones e Instalación de APKs

Esta sección es muy importante para tu seguridad.

  • Verificación: Conexión a GitHub para nuevas versiones.
  • Descarga: El APK se guarda temporalmente en la caché privada.
  • Permiso: Android pedirá permiso para "fuentes desconocidas" solo para actualizar o cargar apps solicitadas por ti.
  • Limpieza: Borrado automático del instalador tras el éxito.

No instalaremos nada sin tu consentimiento explícito.

4. Con Quién Compartimos Información

No vendemos ni alquilamos tu información personal. Solo compartimos datos técnicos con Google AdMob y bajo requisitos legales obligatorios.

5. Seguridad

Tomamos medidas razonables para proteger los datos, aunque ningún sistema es 100% impenetrable.

6. Privacidad de los Niños

No está dirigida a niños menores de 13 años. No recopilamos información personal de niños intencionadamente.

7. Cambios

Podemos actualizar esta política. Se aconseja revisarla periódicamente en esta página.
